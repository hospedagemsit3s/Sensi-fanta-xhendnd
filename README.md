# Fantasensi.ia - Gerador de Sensibilidade para Free Fire

Este projeto contém uma recriação do site Fantasensi.ia, um gerador inteligente de configurações de sensibilidade para o jogo Free Fire.

## Arquivos Incluídos

1. **index.html** - Página HTML principal recriada com estrutura completa
2. **style.css** - Estilos CSS com design moderno e responsivo
3. **script.js** - JavaScript com interatividade e animações
4. **fantasensiia_original.html** - HTML original extraído do site (renderizado pelo React)

## Como Usar

### Opção 1: Abrir Localmente
1. Abra o arquivo `index.html` em qualquer navegador moderno
2. O site funcionará completamente offline

### Opção 2: Servidor Local
```bash
# Usando Python
python3 -m http.server 8000

# Usando Node.js
npx serve .
```

Depois acesse: `http://localhost:8000`

## Características

- ✅ Design responsivo (mobile e desktop)
- ✅ Animações suaves e efeitos visuais
- ✅ 6 categorias de configuração:
  - Sensi dos Famosos
  - Gerar Sensi Aleatória
  - Sensibilidade por Arma
  - DPI (Android)
  - Ciclos (iPhone)
  - Favoritos
- ✅ Efeitos de hover interativos
- ✅ Partículas animadas no fundo
- ✅ Tema escuro otimizado

## Tecnologias Utilizadas

- HTML5
- CSS3 (Gradientes, Flexbox, Grid)
- JavaScript (ES6+)
- SVG Icons

## Observações

- O site original é uma aplicação React/Vite com backend
- Esta versão é uma recriação estática focada no design e layout
- Para funcionalidades completas (geração de sensi, salvamento, etc.), seria necessário implementar backend e lógica adicional
